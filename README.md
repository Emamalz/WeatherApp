# WeatherApp 

WeatherApp is an iOS weather dashboard app built with Swift. It displays weather information in a clean, simple mobile interface.

## Features

- Current weather dashboard
- Forecast display
- Weather icons and visual indicators
- Organised MVVM-style project structure
- Swift-based iOS interface

## Tech Stack

- Swift
- Xcode
- iOS
- SwiftUI

## Getting started

1. Clone the repository
2. Open the project in Xcode
3. Build and run the app using the IOS Simulator
